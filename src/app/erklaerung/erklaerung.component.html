Erklärung

HTML

HTML (Hypertext Markup Language) ist eine textbasierte Sprache und wird zur Strukturierung von Inhalten wie z.B. Texte, Bilder oder Hyperlinks verwendet. Sie hat keine Gestaltung und Formatierung und besteht nur aus Klartext und Elementen (-> Tag). Mit HTML ordnet man diese zusammenwirkende Elemente logisch an.

CSS

CSS wird verwendet für das Layout einer Webseite und ist direkt mit HTML verknüpft. Es beschreibt somit die Darstellung einzelner Elemente. Mit CSS können komplexe und genaue Formatierungen einzelner Tags ermöglicht werden und damit der Inhalt auf verschiedenen Medien flexibel dargestellt werden (-> responsive Webseite) . Ferner dient es der Barrierefreiheit.

JavaScript

JavaScript ist eine dynamisch typisierte objektorientierte klassenlose Skriptsprache. Mit JavaScript wird Dynamik eine eine Webseite gebracht. Dabei werden Webseiten über das Document Object Model (-> DOM) manipuliert. Mit JavaScript kann auf Nutzerinteraktion reagiert werden z.B. bei der Überprüfung von Formularen. JavaScript wird über das script-Tag eingebunden entweder im head-Tag oder im body-Tag.
