Erklärung

HTML

  • Hypertext Markup Language
  • wird zur Strukturierung von Inhalten wie zum Beispiel Texte, Bilder, Hyperlinks verwendet
  • hat keine Gestaltung / Formatierung
  • logische Ordnung von zusammenwirkenden Elementen
  • CSS wird verwendet für das Layout einer Webseite
  • beschreibt die Darstellung einzelner Elemente
  • dynamisch typisierte objektorientierte klassenlose Skriptsprache
  • bringt Dynamik in die Webseite
  • dynamische Manipulation von Webseiten über das Document Object Model
  • beschreibt das Verhalten
  • reagiert auf Nutzerinteraktion
  • Anwendungsbeispiele: Überprüfung von Formularen, Nachladen von Daten (z.B: AJAX)
  • wird über das script-Tag eingebunden entweder im head-Tag oder im body-Tag
